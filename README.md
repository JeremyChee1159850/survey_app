# Green or Mean – Plant Preference Survey Web Application

## Overview
Green or Mean is a web-based survey application developed as part of an industry project with Lincoln University.

The goal of the project is to investigate how aesthetic preferences influence plant selection and whether visually appealing plants contribute to the spread of invasive species in New Zealand.

The system presents users with pairs of plant images and records their preferences and reasoning. The collected data helps researchers understand the factors that influence plant selection behaviour.

## Features

User Features
- Anonymous survey participation
- Side-by-side plant image comparison
- Selection of preferred plant
- Follow-up question asking why the plant was chosen
- Survey completion summary

Researcher Features
- Secure login for researchers
- Upload new plant images
- Manage plant entries
- Expand the survey image database without modifying code
